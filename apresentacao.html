Sobre a Atividade

Esta página foi criada como parte da atividade Receita Culinária em HTML, da disciplina Programação Web I. A proposta da atividade é pesquisar uma receita culinária e construir uma página em HTML contendo menu inicial, links internos por âncoras, imagens da receita, imagens de cada ingrediente e links externos para páginas de compra dos ingredientes.

A receita escolhida foi o Bolo de Milho Cremoso, uma preparação tradicional da culinária brasileira, muito comum em festas juninas. O conteúdo foi organizado de forma simples, utilizando recursos básicos de HTML.

O projeto foi estruturado com uma página de apresentação e uma página principal da receita, permitindo navegação entre as partes do trabalho.

Objetivo

O objetivo principal é praticar a criação de uma página web simples, aplicando tags básicas de HTML para estruturar texto, imagens, listas, tabelas, links externos e navegação interna.

  • Utilizar títulos e subtítulos;
  • Organizar textos em parágrafos;
  • Criar menu com links âncoras;
  • Inserir imagem da receita;
  • Inserir imagem de cada ingrediente;
  • Adicionar links externos para compra dos ingredientes;
  • Organizar informações em listas e tabelas simples;
  • Publicar a atividade em repositório no GitHub.
